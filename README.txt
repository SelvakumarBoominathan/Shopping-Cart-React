This is a simple replication of Ecommerce website.
It has a fucntionality to add/Remove items (as a count) in cart keeper
The layout has been doen using Bootstrap
The buttons have toggle feature. InnerText will change (Add to Cart/ Remove) on a single buttion click.
10 differet items were placed in this website. The data is placed in App.jsx file. 
Each component has its own folder with its own css file in seperate folders. these folders are kept inside "src" folder.
